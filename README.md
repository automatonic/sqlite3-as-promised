# sqlite3-promises

sqlite3 wrapper using q.js promises

## Getting Started
Install the module with: `npm install sqlite3-promises`

```javascript
var sqlite3_promises = require('sqlite3-promises');
sqlite3_promises.awesome(); // "awesome"
```
